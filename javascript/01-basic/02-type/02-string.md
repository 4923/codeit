### 문자열
파이썬처럼 `'` 작은따옴표와 `"` 큰 따옴표 모두 문자열 표기에 사용할 수 있으나 여는 따옴표와 닫는 따옴표는 같아야 한다.

|예|정오|
|:-|:-|
|'작은따옴표쌍'|O|
|"큰따옴표쌍"|O|
|"따옴표 혼용'|X|

### 문자열 활용

- 마찬가지로 back slash를 사용하여 문자열 안의 기호로 인식한다.
- 또는 백틱으로 문자열을 감싸면 문자열 자체를 출력한다.
    ```js
    console.log(`"He said "I'm Iron man"`)
    ```
